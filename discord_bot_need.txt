請幫我建立一個完整可運行的 Python 專案，用於透過 GitHub Actions 定時發送 Discord 訊息。

## 專案目標

每週三晚上 22:00（台灣時間 UTC+8），自動在指定的 Discord 頻道發送一則訊息，然後由 Bot 自己對該訊息新增一個 Emoji Reaction。

此專案將部署在 GitHub Repository 中，透過 GitHub Actions 的 Schedule Trigger 自動執行，不需要任何 VPS 或長駐服務。

---

## 技術要求

### Python

使用 Python 3.12

### Discord

使用 discord.py

### CI/CD

使用 GitHub Actions

### Secrets

所有敏感資訊必須透過 GitHub Secrets 提供，不允許硬編碼於程式碼中。

---

## Discord 功能需求

### 發送訊息

在指定頻道發送訊息：

「大家晚安，這是本週三的自動提醒！」

請將訊息內容抽離成變數，方便未來修改。

### 新增 Reaction

訊息成功送出後，由 Bot 自己新增：

✅

Reaction

流程如下：

1. 發送訊息
2. 取得 Message Object
3. 對該 Message 執行 add_reaction("✅")

---

## GitHub Secrets

程式必須從環境變數讀取：

DISCORD_TOKEN
DISCORD_CHANNEL_ID

範例：

DISCORD_TOKEN=xxxxxxxx
DISCORD_CHANNEL_ID=123456789012345678

不可將 Token 寫死在程式中。

---

## GitHub Actions 需求

建立：

.github/workflows/discord-reminder.yml

### 觸發方式

必須同時支援：

1. 排程執行
2. 手動執行

範例：

on:
workflow_dispatch:

schedule:
- cron: '0 14 * * 3'

說明：

GitHub Actions 使用 UTC 時區

UTC 14:00 = 台灣時間 UTC+8 的 22:00

每週三執行一次
